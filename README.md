# Prop Firm Trading Dashboard

A complete, production-ready personal prop-firm trading dashboard built with React, Vite, Tailwind CSS, and Recharts.

## Features

- **User-Editable Settings**: Account balance, risk %, stop-loss/take-profit pips, phase targets, daily drawdown limit
- **Trade Journal**: Full trade tracking with editable fields and auto-calculated metrics
- **Real-time Calculations**: Equity, win rate, expectancy, strategy grade (A/B/C)
- **Progress Tracking**: Phase-1 and Phase-2 progress bars
- **Equity Curve**: Visual chart using Recharts
- **Data Persistence**: LocalStorage for offline use
- **CSV Export**: Export all trades to CSV format
- **Dark Mode**: Professional prop-firm aesthetic

## Installation

```bash
npm install
```

## Development

```bash

```

